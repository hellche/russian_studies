Версия датасета без фракционализации

Итоговый датасет – это где наблюдениями являются институции. При этом случаи, когда одна статья написана несколькими авторами из одной организации, должны превратиться в случаи, когда этой организации приписана одна статья. Сейчас в файле это не так. То есть, наверное, прежде чем получать файл где институции, нужно в статье, которая множественными строчками записана, убрать все повторные строчки с одной и той же организацией. А потом уже переходить к преобразованию файла, чтобы наблюдения были организациями. В этой версии если статья была написана 2 авторами из разных организаций, то каждой организации приписывается по статье.

  • На примере WOS:A1996WT56000001. 5 соавторов из двух институций. Каждой из двух институции по + 1 остатье от WOS:A1996WT56000001
